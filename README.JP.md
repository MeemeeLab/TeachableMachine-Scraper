# TeachableMachine-Scraper

## TMファイルにパックする機能を備えた簡単かつ強力なスクレイピングツール

[英語](README.md)

![Image](https://i.imgur.com/hl1mxKR.png, "Image")

## 機能
* 複数の言語をサポート
* マニフェストファイルを編集できる
* 画像のスクレイピング
* 画像をパックする (tmファイルに)
* 設定を保存・読み込む

## インストール
1. 最新のNode.jsとnpmをインストール
2. リポジトリをクローン
3. ルートディレクトリで`npm install`を実行
4. ルートディレクトリで`npm start`を実行
